## Sobre el curso

**Python Total** es un curso completo de Python en Udemy que abarca desde los fundamentos del lenguaje hasta conceptos más avanzados, con una duración de **16 días** de contenido.

## Tecnologías y herramientas

- **Lenguaje:** Python 3.x
- **IDE:** PyCharm
- **Plataforma:** Udemy
- **Control de versiones:** Git & GitHub

##  Objetivo

Completar el curso de 16 días adquiriendo una base sólida en Python, desde los conceptos más básicos hasta la programación orientada a objetos y proyectos prácticos reales.
